    Hydriz on GitHub

    I am Hydriz, and this is my website on GitHub. I currently have my code on GitHub, in which you can know more about below.

    Projects

    Looking at my account on GitHub, I operate a few tools that might be of use to you:

    • PrefixExport - Allow for easier exporting of pages in a prefix (MediaWiki).
    • GlobalPageEdit - A python-based Wikipedia bot that allows you to create/edit your global user pages throughout Wikimedia wikis.
    • HostStats - A simple MediaWiki extension for displaying some host statistics about your wiki.

    Archiving projects

    • Dumps - Scripts for archiving database dumps that Wikimedia generates and putting them on the Internet Archive.
    • Incremental dumps - Scripts for archiving the incremental dumps that Wikimedia generates.
    • Incremental tarballs - Scripts for archiving the incremental media tarballs that Wikimedia generates.

    Outside GitHub

    Other than putting my programming work on GitHub, I have a Single User Login (SUL) account throughout the Wikimedia Foundation under the same name. See my userpage on the English Wikipedia for more information.
